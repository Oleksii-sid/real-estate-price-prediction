# Real Estate Price Prediction

Machine learning project for predicting real estate prices using Python and regression models.

## Project Overview

This project analyzes real estate data and predicts housing prices based on different property features.

The goal of the project is to demonstrate:
- data preprocessing
- exploratory data analysis
- machine learning model training
- prediction and evaluation

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Features

- Data cleaning
- Feature selection
- Regression model training
- Price prediction
- Data visualization
